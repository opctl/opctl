Array Parameter [object]

An object defining a parameter which accepts an array typed value.

#Properties:

  • must have:
    • description
  • may have:
    • constraints
    • default
    • isSecret

#constraints

A JSON Schema v4 object defining constraints on the parameters value.

#default

An array to use as the value of the parameter when no argument is provided.

#description

A markdown string defining a human friendly description of the parameter.

#isSecret

An boolean indicating if the value of the parameter is secret. This will cause it to be hidden in UI's for example.
