opctl node create

opctl node create

Start an in-process node which inherits current stderr/stdout/stdin/PGId (process group id) and blocks until interrupted or deleted.

There can be only one node running at a time on a given machine.

Global Options​

see global options

Notes​

lockfile​

Upon creation, nodes populate a lockfile at DATA_DIR/pid.lock containing their PId (process id).

concurrency​

Prior to node creation, if a lockfile exists, the existing lock holder will be liveness tested.

Only in the event the existing lock holder is dead will creation of a new node occur.
